# 三、`soap`

- `SOAP(Simple Object Access Protocol)`简单对象访问协议，一种网络通信协议
- `SOAP`用于跨平台应用程序之间的通信，使用`http`进行通信，允许跨越防火墙
- `SOAP = HTTP + XML`是通过`HTTP`发送`xml`数据
