## 4.1.接口区别

- `JDK7:`只能定义全局常量（public static final）和抽象方法
- `JDK8:`可定义静态方法和默认方法
- `JDK9:`可定义私有方法

### 4.1.1.接口与抽象方法的区别

- 共同点：
  - 不能实例化
  - 以多态的方式使用
- 不同点：
  - 接口
    - 多实现
  - 抽象方法
    - 单继承
