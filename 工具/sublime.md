ctrl + shift + p 面板下的操作

* 设置文档语法 css js 
* 调整缩进 reindent lines,设置快捷键（调整css和js）

```json
{"keys": ["shift+tab"],"command": "reindent","args":{"single_line":false}}
```

* 打开文件属性配置settings 
* 打开快捷键配置 key bindings
